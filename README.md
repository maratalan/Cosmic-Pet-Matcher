# Cosmic-Pet-Matcher
I have built the Cosmic Pet Matcher, a mystical application that aligns an individual’s zodiac energy with the perfect animal companion.
<div align="center">
This is my capstone project for the Google AI Agents: Intensive Vibe Coding Course.
I have built the Cosmic Pet Matcher, a mystical application that aligns an individual’s zodiac energy with the perfect animal companion.
The Cosmic Pet Matcher includes the Gemini Studio AI image generation and vibe coding capabilities to create unique, mystical portraits of the user’s destined companion.  In short, your Horoscope and Lifestyle Determine the ideal pet.  Asking for the users zip code, we then provide updated links to sites like Petfinder that in turn work with shelters to match you with the ideal pet.  Dogs, cats, rabbits, snakes, etc.
Cosmicpet takes the broad strokes of the horoscope as a frame for personality, asks questions as to preferences in terms of activity level, sedentary vs. active, time available to work with the pet.  I have upgraded the Cosmic Pet Matcher with advanced AI image generation to create unique, mystical portraits of your destined companion.
Key Features:
•	Celestial Interface: An immersive, atmospheric design featuring radial gradients, glassmorphism, and smooth transitions powered by motion.
•	Zodiac Alignment: A beautifully crafted selector for all 12 zodiac signs, each paired with symbolic icons representing their elemental nature.
•	AI-Powered Matching: Uses Gemini 3 Flash to analyze your zodiac sign's personality traits and determine the ideal match among four distinct pet categories: Dogs, Cats, Snakes, and Gerbils.
•	Mystical Explanations: Each match comes with a detailed, markdown-formatted cosmic explanation of why that specific animal is your destined soul companion.
•	Responsive & Polished: Designed to feel like a premium editorial experience on both desktop and mobile devices.
Technical Details:
•	Frontend: React 19 with TypeScript.
•	Styling: Tailwind CSS 4.0 with a custom "Cosmic" theme.
•	Animations: motion (formerly Framer Motion) for layout transitions and entrance effects.
•	AI Integration: @google/genai SDK with structured JSON output for precise pet matching.
•	Icons: lucide-react for a consistent, modern iconography set.

Tech Stack:
Python | Google Gemini API | JSON/CSV | Git & GitHub

Mission:
Working with shelters in different countries, I found that there is often one overarching situation that leads people to return the pet: We think we know the pet we want but lifestyle, personality and failed expectations often lead to abandonment.
This inspired me to build CosmicPet- Find your petmate for your lifestyle, an AI powered platform that acts matchmaker using Horoscope as the basis.
